##  html相关操作函数

```go
import "github.com/x-module/helper/html"
```

#### 函数列表:

-   **<big>Html2JS</big>** converts []byte type of HTML content into JS format
-   **<big>HtmlEncode</big>** html chars to string
-   **<big>HtmlDecode</big>** decodes string to html chars.
-   **<big>StripTags</big>** tags in html string.
-   **<big>Nl2br</big>** change \n to <br/>.
